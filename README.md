# Eshika Agnihotri - Portfolio

This is a 3D responsive portfolio website showcasing my resume, skills, achievements, and projects.

## 🚀 Features
- Responsive design (mobile + desktop)
- 3D hover effects on skills
- Smooth scrolling navigation
- Contact form

## 📂 Sections
- Hero (Name + Contact)
- About Me
- Education
- Skills
- Achievements
- Experience
- Contact

## 🌐 Deployment
1. Fork this repo
2. Push to GitHub
3. Enable GitHub Pages in repo settings
4. Access via `https://username.github.io/portfolio-eshika`
